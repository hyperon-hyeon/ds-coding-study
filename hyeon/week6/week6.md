## 1. 종이의 개수 - 백준 1780번
    

### ❓문제 요약

- **목표**: - NxN 행렬이 같은 숫자(-1,0,+1)로 채워져 있는지 확인하고 개수 세기
    1. 만약 종이가 모두 같은 수로 되어 있다면 이 종이를 그대로 사용한다.
    2. (1)이 아닌 경우에는 종이를 같은 크기의 종이 9개로 자르고, 각각의 잘린 종이에 대해서 (1)의 과정을 반복한다.

- **자료구조**: 분할정복 알고리즘
- **입력**
    - 첫째 줄: 줄 수 N
    - 둘째 줄부터: N개의 줄에 N개의 정수 행렬
- **제약 조건**
    - 1 ≤ N ≤ 37, N은 3k 꼴
- **출력**
    - 첫째줄에는 -1로 채워진 종이의 개수, 둘째줄에는 0으로 채워진 종이의 개수, 셋째줄에는 +1로 채워진 종이의 개수 수를 출력

### ❗코드 설명

**변수**

- count 배열: -1,0,+1 개수를 세어 저장하는 배열
- paper 배열: 입력받은 행렬 저장

**코드 설명**

1. N을 입력받고 paper 배열 초기화한 뒤 저장
2. divideAndConquer 함수 호출
    - Check 함수 호출하여 같은 숫자로 이루어져 있는지 확인 후 boolean 값 반환
        - true라면 count 증가 후 종료
        - false라면 size를 3으로 나누고 새로운 조각 좌표 계산 후 divideAndConquer 재귀 호출
3. 결과 출력 - count 배열 출력

## 2. Z - 백준 1074번
### ❓문제 요약

- **목표**: 크기가 2^N × 2^N인 2차원 배열에서 r행 c열을 언제 방문했는지 구하기
    - N > 1인 경우, 배열을 크기가 2^(N-1) × 2^(N-1)로 4등분 한 후에 재귀적으로 순서대로 방문
- **자료구조**: 분할정복 알고리즘
- **입력**
    - 첫째 줄: 크기 N, 구해야 하는 행과 열(r, c)
- **제약 조건**
    - 1 ≤ N ≤ 15
    - 0 ≤ r, c < 2^N
- **출력**
    - r행 c열을 몇 번째로 방문했는지 출력

### ❗코드 설명

**변수**
- result: 결과 저장 변수

**코드 설명**
1. N, r, c 입력 받음
2. findZOrder 함수 호출
    - 목표 좌표가 현재 사분면으로 분할(크기 절반)하고 시작점 기준 어디에 속하는지 판단함
    - 목표지점이 없는 사분면은 건너뛰고 그만큼 result에 더함
    - 위의 단계를 반복하면서 점차 작은 사이즈로 줄이다가 size가 1이 되면 중단
3. 결과 출력 - result 값 저장
