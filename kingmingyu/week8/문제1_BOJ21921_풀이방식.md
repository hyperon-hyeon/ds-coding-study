# 문제
찬솔이가 N일동안 블로그를 하는데 N일 중에서 X일동안 가장 많은 방문자수와 횟수를 반환하는 프로그램 작성하기

## 슬라이딩 윈도우 이용
전체 N일 동안 고정된 X일 중에서 가장 많은 방문자수와 횟수를 출력하기 때문에 슬라이딩 윈도우를 이용

## 새로운 윈도우 생성
```java
int sum = 0;
// 새로운 윈도우 생성
for(int i = 0; i < X; i++){
    sum += visit[i];
}
```

## 윈도우를 이동시키면서 가장 많은 방문자 수와 횟수 업데이트
```java
int answer = sum; // 처음 생성한 윈도우 값 넣어주기
int count = 1;
// 윈도우 이동 + count, answer 갱신
for(int i = X; i < N; i++){
    sum = sum - visit[i - X] + visit[i];
    if(sum > answer){
        count = 1;
        answer = sum;
    }
    else if(answer == sum){
        count ++;
    }
    //System.out.println("i: " + i + " answer: " + answer + "count: " + count);
}
```

## 정답 출력
```java
//정답 출력
if(answer == 0){
    System.out.println("SAD");
}
else{
    System.out.println(answer);
    System.out.println(count);
}
```
