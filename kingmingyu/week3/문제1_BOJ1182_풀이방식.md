# 문제
입력받은 숫자 배열을 조합하여 특정 정수를 만들 수 있는 조합의 개수 출력하기

## 자료구조
```java
static int[] arr;
static int n;
static int s;
static int cnt = 0;
```
- `arr[]`: 숫자 배열을 입력받을 arr
- `n`: 배열의 개수
- `s`: 조합하여 만들고자 하는 정수
- `cnt`: 조합의 개수

## dfs & 백트래킹
```java
public static void dfs(int d, int sum){
    if(d == n){
        if(sum == s){
            cnt++;
        }
        return;
    }

    //현재 원소를 선택하는 경우
    dfs(d + 1, sum + arr[d]);
    //현재 원소를 선택하지 않는 경우
    dfs(d + 1, sum);
    }
```
- 배열에서 원소를 선택하는 경우
- 배열에서 원소를 선택하지 않는 경우 모두 탐색
- d == n인 경우 즉 원소를 모두 탐색한 경우 sum을 확인

```java
if(s == 0) cnt--; // 공집합 제외
```
추가로 문제에서 공집합인 경우는 제외하므로 조합하여 만들고자 하는 숫자가 0인 경우 공집합이 포함되므로 `cnt--`
