# 문제
스도쿠 빈칸 채우기

## 1. 빈칸에 들어갈 숫자 탐색
```java
public static boolean isTrue(int r, int c, int value){
    //세로 줄 확인
    for(int i = 0; i < 9; i++){
        if(sudo[i][c] == value){
            return false;
        }
    }

    //가로 줄 확인
    for(int i = 0; i < 9; i++){
        if(sudo[r][i] == value){
            return false;
        }
    }

    //같은 칸 확인
    int cellR = (r / 3) * 3;
    int cellC = (c / 3) * 3;
    for(int i = cellR; i < cellR + 3; i ++){
        for(int j = cellC; j < cellC + 3; j++){
            if(sudo[i][j] == value){
                return false;
            }
        }
    }

    return true;
}
```

## 2. dfs + 백트래킹
```java
public static void dfs(int r, int c){
    if(c == 9){
        dfs(r + 1, 0);
        return;
    }

    if(r == 9){
        for(int i = 0; i < 9; i++){
            for(int j = 0; j < 9; j++){
                System.out.print(sudo[i][j] + " ");
            }
            System.out.println();
        }
        System.exit(0); //가장 처음 답만 찾고 종료
    }

    //0이 아닌 경우는 pass
    if (sudo[r][c] != 0) {
        dfs(r, c + 1);
        return;
    }

    //0인 경우 채울 수 있는 값을 찾아 채우고 넘어가기
    for(int i = 0; i < 9; i++){
        if(isTrue(r, c, i + 1)){
            sudo[r][c] = i + 1;
            dfs(r, c+1);
        }
    }
    
    sudo[r][c] = 0;
}
```

- 탐색 방법: c값을 1씩 늘리며 가로줄을 탐색하고 c값이 9가 되었을 때 c = 0, r += 1을 통해 다음 줄을 탐색
```java
if(c == 9){
    dfs(r + 1, 0);
    return;
}
    
dfs(r, c+1);
```

- r값도 9가 된 경우(스토쿠에서 가로 세로 모두 0 - 8까지 탐색한 경우) 답안을 출력하고 종료
```java
if(r == 9){
    for(int i = 0; i < 9; i++){
        for(int j = 0; j < 9; j++){
            System.out.print(sudo[i][j] + " ");
        }
        System.out.println();
    }
    System.exit(0); //가장 처음 답만 찾고 종료
    }
```

- 주어진 스토쿠 칸의 값이 0이 아닌 경우
```java
if (sudo[r][c] != 0) {
    dfs(r, c + 1);
    return;
}
```

- 주어진 스도쿠 칸의 값이 0인 경우
```java
for(int i = 0; i < 9; i++){
    if(isTrue(r, c, i + 1)){
        sudo[r][c] = i + 1;
        dfs(r, c+1);
    }
}
```

- 만약 1~9까지 다 넣어봤는데 길이 막혔다면(해당 경로에서 답이 없다면) 
- 현재 칸을 다시 0으로 만들고 이전 재귀로 복귀 (백트래킹)

    `sudo[r][c] = 0;`
