# 문제
2 x n의 타일에서 2 x 1의 직사각형으로 채울 수 있는 경우의 수 출력하기

## 풀이
1 -> 1

2 -> 2

3 -> 3

4 -> 5

...

9 -> 55

위 수열의 형태는 피보나치 수열의 형태를 따른다.

## 피보나치
```java
public static int fib(int n){
    if(n == 1) return 1;
    if(n == 2) return 2;

    int sum = 0;
    int f1 = 1;
    int f2 = 2;

    for(int i = 2; i < n; i++){
        sum = (f1 + f2) % 10007;
        f1 = f2;
        f2 = sum;
    }
    return sum;
}
```
- `n = 1`인 경우와 `n = 2`인 경우를 설정
- 반복문의 형태로 피보나치 수열 구하기

## 전체 코드
```java
package kingmingyu.week9;

import java.io.BufferedReader;
import java.io.IOException;
import java.io.InputStreamReader;

public class 문제1_BOJ11726_풀이 {
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        int n = Integer.parseInt(br.readLine());

        System.out.println(fib(n));
    }

    public static int fib(int n){
        if(n == 1) return 1;
        if(n == 2) return 2;

        int sum = 0;
        int f1 = 1;
        int f2 = 2;

        for(int i = 2; i < n; i++){
            sum = (f1 + f2) % 10007;
            f1 = f2;
            f2 = sum;
        }
        return sum;
    }
}
```
