# 문제
도현이의 집 N개가 수직선 위에 존재 (각각의 집의 좌표는 x1, ..., xN, 집 여러개가 같은 좌표X)

집에 공유기 C개를 설치하려고 할 때 가장 인접한 두 공유기의 사이의 거리를 가능한 크게 하여 설치(같은 집에는 1개만)

-> C개의 공유기를 N개의 집에 적당히 설치, 가장 인접한 두 공유기 사이의 거리를 최대로 하는 프로그램을 작성

## 알고리즘
두 공유기 사이의 일정한 거리를 최대로 하는 거리 구하기

-> 거리를 반 씩 줄여가며 탐색 (bnc)
```java
//공유기 설치 최소 거리
int min = 1;
//공유기 설치 최대 거리
int max = homeX.getLast() - homeX.getFirst() + 1; // 백준 제출 시 .get으로 바꾸기

while(min < max){
    //탐색 값 (반 씩 줄여가며 탐색)
    int mid = (min + max) / 2;

    //설치된 공유기의 수가 주어진 공유기 보다 적다면 반 줄이기
    if(install(mid) < C){
        max = mid;
    }
    //설치된 공유기의 수가 크거나 같다면 정답이거나 더 넓게 설치할 수 있으므로 +1하여 탐색
    else{
        min = mid + 1;
    }
}
```

- 거리를 입력 받았을 때 설치 가능한 공유기의 수 반환
```java
public static int install(int dist){
    int count = 1;
    int lastLocate = homeX.getFirst();

    //mid 거리로 공유기 설치
    for(int i = 0; i < homeX.size(); i++){
        int locate = homeX.get(i);

        if(locate - lastLocate >= dist){
            count++;
            lastLocate = locate;
        }
    }
    //설치된 공유기의 수
    return count;
}
```

- 정답 출력
- while문 종료 조건에 의해 max값과 min값이 같아질 때 종료되고 그 값이 최소 거리 +1 이므로 -1하여 출력
```java
System.out.println(min - 1);
```
