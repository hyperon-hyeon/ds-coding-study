# 문제
n자리 숫자에서 k개의 숫자를 지웠을 때 나올 수 있는 가장 큰 수 구하기

1. 가장 작은 숫자 지우기 먼저 지우기
    
    -> 4177252841 처럼 중요하지 않은 뒷자리에 작은 숫자가 나온 경우 지울 필요 없음 (X)
2. 앞에서부터 뒷 자리 숫자보다 작은 경우만 지우기

## 자료구조
- stack을 이용하여 현재 숫자와 비교하여 삭제 여부 판단

```java
int now = str.charAt(i) - '0';

// 현재 숫자가 스택 마지막보다 크면 이전 숫자 제거
while (!stack.isEmpty() && k > 0 && stack.peekLast() < now) {
    stack.pollLast();
    k--;
}
```
- 현재 숫자가 스택 마지막보다 크면 이전 숫자를 제거

```java
// 아직 제거 안 한 k가 남으면 뒤에서 제거
while (k-- > 0) {
    stack.pollLast();
}
```
- 제거해야 할 숫자(k)가 남아있다면 뒤에서부터 제거

# 전체 코드
```java
package kingmingyu.week5;

import java.io.BufferedReader;
import java.io.IOException;
import java.io.InputStreamReader;
import java.util.ArrayDeque;
import java.util.Deque;
import java.util.StringTokenizer;

public class 문제2_BOJ2812_풀이 {
    public static void main(String[] args) throws IOException {

        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        StringTokenizer st = new StringTokenizer(br.readLine());

        int n = Integer.parseInt(st.nextToken());
        int k = Integer.parseInt(st.nextToken());
        String str = br.readLine();

        Deque<Integer> stack = new ArrayDeque<>();

        for (int i = 0; i < n; i++) {
            int now = str.charAt(i) - '0';

            // 현재 숫자가 스택 마지막보다 크면 이전 숫자 제거
            while (!stack.isEmpty() && k > 0 && stack.peekLast() < now) {
                stack.pollLast();
                k--;
            }
            stack.addLast(now);
        }

        // 아직 제거 안 한 k가 남으면 뒤에서 제거
        while (k-- > 0) {
            stack.pollLast();
        }

        // 출력
        for (int num : stack) {
            System.out.print (num);
        }
    }
}
```
