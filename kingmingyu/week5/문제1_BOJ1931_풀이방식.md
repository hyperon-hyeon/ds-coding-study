# 문제
가장 많은 회의실을 배정하기

1. 가장 먼저 시작하는 회의 배정

    -> 가장 먼저 시작하고 가장 늦게 끝나는 경우(X)
2. 가장 짧은 시간의 회의 배정
    
    -> 회의 시작 시간 앞 뒤로 다른 회의가 배정된 경우(X)
4. 가장 빨리 끝나는 회의 배정


## 가장 빨리 끝나는 회의 선택
```java
room = new int[n][2];
```
- 회의의 시작 시간 room[n][0]과 회의의 끝나는 시간 room[n][1]을 저장할 2차원 배열

```java
Arrays.sort (room, new Comparator<int[]>() {
    @Override
    public int compare(int[] o1, int[] o2) {
        if(o1[1] == o2[1]){
            return o1[0] - o2[0];
        }
        return o1[1] - o2[1];
    }
});
```
- 회의가 빨리 끝나는 것을 기준(room[n][1]이 작은 것)으로 정렬

    -> `return o1[1] - o2[1];`
- 만약 회의가 끝나는 시간이 같다면 시작하는 시간이 짧은 회의(room[n][0]이 작은 것)으로 정렬

    -> `return o1[0] - o2[0];`

```java
for(int i = 0; i < n; i++){
    if(cur_end_time <= room[i][0]){
        cur_end_time = room[i][1];
        count++;
    }
}
```
- 문제 조건에 맞는지 확인(feasibility check)
- 현재 회의가 끝나는 시간이 회의가 시작하는 시간보다 작으면 회의실 배정 + 끝나는 시간 갱신

## 전체 코드
```java
package kingmingyu.week5;

import java.io.BufferedReader;
import java.io.IOException;
import java.io.InputStreamReader;
import java.util.Arrays;
import java.util.Comparator;
import java.util.StringTokenizer;

public class 문제1_BOJ1931_풀이 {

    static int[][] room;

    public static void main(String[] args) throws IOException {

        BufferedReader br = new BufferedReader (new InputStreamReader(System.in));
        int n = Integer.parseInt (br.readLine ());

        room = new int[n][2];

        for(int i = 0; i < n; i++){
            StringTokenizer st = new StringTokenizer (br.readLine ());
            room[i][0] = Integer.parseInt (st.nextToken ());
            room[i][1] = Integer.parseInt (st.nextToken ());
        }

        Arrays.sort (room, new Comparator<int[]>() {
            @Override
            public int compare(int[] o1, int[] o2) {
                if(o1[1] == o2[1]){
                    return o1[0] - o2[0];
                }
                return o1[1] - o2[1];
            }
        });

        int count = 0;
        int cur_end_time = 0;

        for(int i = 0; i < n; i++){
            if(cur_end_time <= room[i][0]){
                cur_end_time = room[i][1];
                count++;
            }
        }

        System.out.println (count);

    }
}

```
